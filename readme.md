# 开发文档

## 一、文档说明

- 在使用K210的过程中，存在各种各样的问题。而如何将K210接入局域网，进行图像传输，是这次项目的核心，但是，由于k210的核心以及源码部分不予开源，而我们采用的W5500并不是在官方店铺进行购买的。所以次次项目就是为了打通k210的spi，使其能够正常与W5500进行通信



## 二、文档结构

- **img** 为开发过程中的截屏文件
- **inc** 为资料文件
- **src** 为核心代码
